# Apple Liquid Glass Design Mechanics (raw extraction)

## Core Material Properties
- Translucent, reflects and refracts surroundings
- Color informed by surrounding content — contextually adaptive
- Real-time specular highlights (respond to device motion)
- Adaptive shadows
- "Intelligent rendering" switches between light and dark environments

## Depth Mechanics
- Sidebars refract content behind them while reflecting wallpaper/surroundings
- Controls as "distinct functional layer that sits above apps"
- Border replacement: controls "fit perfectly concentric with rounded corners of hardware" — no flat border, instead the curve of the material itself defines the edge

## State Transitions
- Active vs inactive via "dynamic morphing" — controls change shape, not just color
- Tab bars shrink when scrolling (focus content), expand when scrolling back (restore navigation)
- State encoded in geometry/shape, not just color/opacity

## Announced
- WWDC June 9, 2025
- Applies across iOS 26, iPadOS 26, macOS Tahoe 26, watchOS 26

## What This Means for Canvas Design
- Status signals can be depth/shape-based rather than color/border-based
- Translucency as connection: seeing through elements creates contextual grounding
- Dynamic shrink/grow for active elements = physics-adjacent interaction
- Material as identity: the material properties ARE the brand, not overlaid on it
